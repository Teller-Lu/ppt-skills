## 一句话区分"成品" vs "过程"
- **进 repo / 是 skill**:`skills/ppt-skills/`(SKILL+scripts+references+templates+styles+components)、`.claude-plugin/`、`docs/`、`README`。
- **不进 repo / 纯过程**:`_inbox/`(源模板)、`_experiments/`(真图、工具、deck 产物)。

## 学一个新模板的标准流程(用 `_toolkit/`)
1. 模板放 `_inbox/<名>.pptx`。
2. `render_truth.ps1` 渲真图 → **看 PNG 学母题/技法**(绝不看有损 SVG)。
3. `strip_dlp.py` 去保密章;`run_convert.py` / `analyze_slides.py` 出结构与校准 → 写 `styles/<名>/`。
4. 据真图建 / 复用组件(真图标 + 母题)→ `aesthetic_checker` 过非朴素门。
5. `full_export.py` + `render_deck.ps1` 导出并回渲核验。
