## 〇、学习一页模板的标准流程（每页照走，别压缩）

学一页 = **10 步**，少一步都算没学透（2026-06-17 用户纠：我曾把它压成"存素材 + 设计"两步；2026-06-18 用户加收尾两步 9/10）：
> **每开学一页前(2026-06-20 用户加)**：① `/compact` 压上下文（学一页周期长、上下文易膨胀）；② **压缩后第一件事 = 重申并坚持"全程中文"**——回复 / 注释 / 脚本输出全中文；实测压上下文后这条最易被丢、混入英文，所以开工先自我重锚（亦见记忆 [[chinese-only]]）。
1. **文字解读**模板那页：大方向 + 细节，内容 / 风格 / 特征，纯描述模板、不掺自己。
2. **用户审解读**（看我注意到什么、漏了什么）——这是个**门**，别跳过直接建。
3. **据反馈校准**认知。
4. **存矢量素材（"存素材标准" 5 步**，每次**全做**，别只做 ①②——否则 palette / preview 里看不到 = 等于没产出，2026-06-17 用户纠）：
   - 🔴 **铁律·只准调库别内联（2026-06-22 用户纠）**：凡可复用图形母题（手机样机 / 卡 / 球 / 装饰块 / 大圆 / 面板…）**必须 distill 入库、builder 只 `place()` 调库**；**严禁在 builder 里内联写死 substantial 图形 XML**——p99 我把手机/车/背景内联画了、高清车图也没存，被纠"调用只能调用素材，不能用记忆"。判据：图形若「换页还想用」或「用户会想换 / 填」，它就是**资产**、进库；只有纯版式定位（对齐 / textbox / 文字）才许 builder 现写。亦见 [[ppt-incremental-assets]]、catalog。
   - ① 抽原生 XML → `vectors/<bz-名>.xml`（去 © / DLP 水印）。**「自带文字槽位的母题」（`grpSp` 内节点 / 编号 / 标签）保留文字 run**（清水印、留 run 的位置 + 字号 + 颜色），回填只改 `a:t` 内容——**别清空文字后按子件 off 叠新 textbox**：`grpSp` 若 `off/ext ≠ chOff/chExt`（被缩放 / 平移），子件「组内坐标」≠画布坐标，手叠必偏移（p30 节点偏 0.6 寸教训，§6.22）。**字体必烤进 run**：所有文字 run 显式写 `<a:latin/ea/cs typeface="微软雅黑">`（**`ea` 管中文、必设**），别靠主题字体——素材注入新 pptx 时宿主主题 `ea` 多为空 → 中文**回退宋体**（PPT 大忌，与 schemeClr 泛色同类坑，§6.23）；python-pptx `font.name` 只写 `latin` 不写 `ea`，自画文字也要手补 `ea`；数字可留 Bebas（保 `latin`、`ea` 仍微软雅黑）。
   - ② 写 `catalog.json`（id / file / **method①②③④** / **inject** / role / tags / effect / place / adjust / color_basis / source）。
   - ③ 进 `_palette.pptx`（每件一页 + 文字说明）：**自然比例摆，文字类素材字号随形状缩**（§六.13）；**白 / 半透件**用【幻灯片背景填充】垫深色对比底——**不是加矩形形状**（形状会被连同素材一起复制走、且只盖一半，2026-06-17 用户纠）；**`scene3d` 件**脚本只能烘焙**缩略图（死图、只能看不可编辑）**——要**可编辑的矢量+文字版**进 palette，**只有 PowerPoint 手动复制粘贴**能做到（脚本注入即空），故这类件在 palette 留**手动粘贴位 + 红字注记**、调用时直接 clone 模板那页；**失败 / 重复的捕获从 palette + catalog + vectors 一并删掉**（小而精，2026-06-17 用户纠）。
   - ⑥ **可见性检查（写进流程·必做）**：④ 渲染后**逐张看 `_preview`**，确认每件**肉眼可见**——白 / 半透件是否已垫深底、`scene3d` 件是否已用缩略图；任何"框选得到但看不见 / 整页空白"的，回 ③ 修到可见为止。
   - ④ 重渲 `_preview/*.png`：**只渲新增 / 改动的那一页**——`render_slides.ps1 -Pptx _palette.pptx -Slides "<页号>" -OutDir _preview -Prefix slide -Width 1280`（非破坏、不删别张）。**绝不用 `render_truth.ps1` 对 `_preview` 全量重渲**——它 `Remove-Item OutDir\*` 把全部 png 删光重生、冲掉手工调好的深底（2026-06-21 用户纠:浪费 token + 毁格式）；`render_truth.ps1` 已设护栏禁止指向 `_preview/_gallery/assets`。
   - ⑤ 工具：`sync_palette_add.py <id>` **增量**追加新件（不重建、保住用户手粘页）→ 再按 ④ **只渲那一页**。**绝不整册重建 `_palette.pptx`**（会冲掉手粘页 / 深底，2026-06-21 用户纠）。白 / 半透件的深底见 ③ 用**幻灯片背景填充**（`fix_palette_bg.py` 可批量设），别加矩形。
   > **图片件例外**（2026-06-18 定）：图片（`rasters/*.png`）只做 **① 存 `rasters/` + ② 入 `catalog`**，**跳过 ③④⑤（不进 `_palette` / `_preview`）**——图片从 `rasters/` 调用、且本身可直接 Read 看，无需 palette 显形。故 **catalog（全）件数 > palette（只矢量+scene3d）页数是正常的**（见 [`资产地图.md`](资产地图.md)）。
5. **自己设计生成**：版式我定 + 每元素四选一出件（§4：自画 / 素材 / clone / 烘焙）。**图标来源**：通用图标一律取 `templates/icons`（tabler/phosphor…≈7981 个，库里现成、不蒸进矢量库 §6.17）；原生注入用 `svglib` 把 SVG recolor 后栅格化成 PNG 贴进**白底容器**（白圆/白环心→白底图标边缘隐形、免透明）。
6. **渲染 + §5 自检**：**必量模板真实字号**（XML `sz`÷100）对齐，逐条过清单。
7. **用户反馈**。
8. **提炼收获 → 更新本文件 / skill**：这页学到的配方、规则、坑都沉淀，别只做不记。
9. **归档成果（2026-06-18 加；2026-06-20 必做；2026-06-21 用户纠·gallery 收敛单一册）**：`_gallery/` 只留**一个**汇总册 `business-blue-master.pptx`（**唯一可编辑 pptx = 单一真相**；用户要 **同一风格/同一阶段所有页合进同一个 pptx**，完整框架 封面→…→结尾；**另起新册用户会专门说**）+ 每页一张 png（供人快速直观看）。**不再为每页留单独 `.pptx`**（2026-06-21 用户:单独 pptx 完全没必要）。学完一页 → ① builder 把成品建到**临时 pptx**（`_experiments/`，用完即删）；② `add_gallery_page.ps1 -Temp <临时> [-At <位> -Replace]` 插进汇总册；③ `sync_gallery_png.ps1 -Only "<页号>"` **从汇总册**渲该页 png（png 永远从汇总册派生 → 不脱钩；并更新脚本 `$NAMES`）。中间版不进 gallery。（教训:① 原生重做期只做散件没维护合集册;② 2026-06-21 png 与汇总册脱钩[showcase.png 旧] + 单独 pptx 冗余,用户纠。）
10. **清理临时（2026-06-18 加；2026-06-20 用户纠·必做）**：只删本页 `_experiments/<page>_design/` 下的**临时件**（探查/蒸馏/构建/诊断脚本、stage 快照、中间 pptx、`_ic_`/`_tmp_` 图标、preview 目录），成果已在 §9 留底。**绝不删成果/工具**——`_gallery/`（散件源页 + 合集册）、`_experiments/_toolkit/`、`assets/`、`references/` 一律**排除在清理之外**；只动 `<page>_design/`。**血的教训**：原生重做期把旧 SVG 合集 `biz_deck/` 当临时清掉、又因 ppt-skills 无 git 兜底而**永久丢失**（2026-06-20 用户纠）。⚠ `render_truth.ps1` 渲染后 COM 不 Quit（避免关用户的）→ 可能**滞留句柄锁住空目录**：内容物删净即可，**空壳被锁就下次会话开头补删**，别去关用户的 PowerPoint。**另：跑脚本别 `cd` 进 `<page>_design`**——Bash shell cwd 持久，会锁住该目录致整目录「in use / Device busy」删不掉（p30 / p53 都踩、留空壳）；一律**用绝对路径**跑脚本，或清理前先 `cd /` 出去。
> 我最常出事处：跳过 2（不等审就建）、跳过 8（学完不沉淀）、做砸 6（不量字号就拍脑袋）、漏 9/10（不归档 / 不清临时）、**把母题内联画而不入库（p99 教训）**。盯死。
> **字号刻度**：按 [`design-language.md`](design-language.md) §1.3 的**统一字号刻度**（内容页标题/引导 28 粗、正文≈14、签名页主标按模板实测放大、硬下限 14）；蒸 / 渲染时**先量模板真值**对齐，别靠手感。
