# business-blue · 审美 DNA(品牌地板)

> 提炼自 `work_inspire_128`(工作型2.0 · 128 页蓝色商务汇总册)真渲染。
> 第二块 floor,与 `academic-defense` 并列;高度统一的蓝色商务系。

## 配色
| 角色 | 色值 | 用途 |
|---|---|---|
| 背景 bg | `#F4F7FC` / `#EBF2FD` / `#F0F6FE` / `#FFFFFF` | 白底冷蓝,正文页主底 |
| 主蓝 primary | `#045FD4` / `#0A5FD4` / `#0275FE` | 主色块/按钮/标题底/强调条 |
| 深蓝 deep | `#012C6D` / `#003891` / `#054DAB` | 章节满底反白/页眉/深色区 |
| 亮天蓝 accent | `#079FFE` / `#0A9FFE` / `#28A1FF` | 图标/数字/重点,招牌色 |
| 青 cyan | `#3FC0FF` / `#54CCFF` / `#63C7FE` | 渐变末端/次强调 |
| 长春花 periwinkle | `#4D78DF` / `#6990E6` / `#95AFEC` | 封面柔和渐变 |
| 浅蓝填充 tint | `#DCEBFB` / `#ACD2F2` / `#ACCCE9` / `#E1EFFC` | 卡片/色块浅底 |
| 文字 ink | 主 `#1F2733` · 次 `#5C636E` · 弱 `#98A0AC` | 三级文字 |

**招牌渐变**:`accent→cyan`(`#079FFE → #54CCFF`)—— 图标圆盘 / 强调条 / 大数字。章节用 **deep navy 实底反白**。封面用 **periwinkle 柔和渐变**。

## 字体(安全优先)
- **中文正文/标题**:微软雅黑 Microsoft YaHei(模板实际主力,217 次;Windows 自带 = 安全)。
- 中文标题加重:微软雅黑 Bold(模板原用 思源黑体 / HarmonyOS Sans SC / OPPOSans,需安装 → 回退雅黑)。
- **拉丁/数字**:Roboto / Arial(安全)。
- **大号装饰数字**:模板用 Bebas(condensed,非系统字体)→ **不直接排,大数字烘焙成 PNG** 或退 Arial 加粗。
- 纪律:仅安全字体直排;非安全显示字体的大字一律烘焙(见 shared-standards)。

## 字号体系(1280×720;**2026-06-15 按模板真实尺度重校**,见记忆 ppt-readable-scale)
`[90, 64, 40, 34, 28, 24, 20, 17, 14]`
- 封面主标 ~64–92 · **章节标题 ~66 / 章节 PART 眉标 ~40**(p19 实测) · 章节大编号(3D) ~233 · **页标题 ~34** · 大陈述/枢纽中心 ~28–33 · **卡片/条目标题 ~24–27** · 子标题/职务/眉标 ~18–20 · **正文 ~17–18** · 标签/轴/注释 ~14–15。
- ⚠️ **签名页(封面/目录/章节)主标题别套内容页的 34**——按模板实测放大(如章节标题 66),量 `sz`÷100,生成后核字号(2026-06-17 用户纠"字偏小")。
- **硬下限 14(<14 美学门 FAIL)**;严禁凭"AI 手感"再缩。早期初值(页标题 26-30/正文 13-14/标签 10.5)**偏小,已废弃** —— PPT 给人汇报,字号至少对齐模板。
- 占版:中心母题/卡片/图标**放大铺满安全区**,勿缩中央留大白(美学门 `min_fill_w/h_ratio` 把关)。

## 间距 / 栅格
- 安全边距 ≈ 左右 64 / 上下 44。
- 页标题锚点 ≈ 左 76 / 上 64(批A 核定)。
- 卡片圆角 ≈ 10–14;卡片间距 ≈ 24;信息密度中等偏高(工作汇报风)。

## 边界系统与凸出节奏(占版)
- 安全边距 ≈ 左右 64 / 上下 44(自动门用)。模板**视觉内容带**更靠内:左右 ≈ 1/15(~85px)、上 ≈ 1/6(~120px)、下 ≈ 1/9(~80px)——标题/卡片重心落此带内。
- **凸出节奏**:绝大多数元素守边界,仅 1–2 件装饰(蓝球)受控越界出血——"既有规矩又有凸出"。生成判断见 `../../references/design-language.md` §1.3–1.4。

## 主题色统一(一改全改)
- 全套由**主题色 accent1 `#0165FF` + accent2 `#00BEFF`**(主题层,`schemeClr` 的解析基准)两色驱动:球的径向渐变、招牌渐变、连城市/云的 **duotone** 都基于 accent1。
- 暗面用 `schemeClr lumMod 64%`(= 亮度 −36%)。
- 改这 2 个主题色 = 整体换风格;素材保留 `schemeClr` → 一键换色。

## 调性
专业、克制、科技感。白底为主 + 蓝色信息块 + azure→cyan 图标圆盘;章节深蓝满底反白;封面柔和渐变。比学术蓝更"亮、更azure、更密"。
