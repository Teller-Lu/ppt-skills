## 逐页块（每页一节，复制下面这块往下排）

```text
### P{n} {页标题}
- 角色：封面 / 过渡 / 内容 / 结尾
- ① 一句话：____            ← 这页让听众记住的唯一核心；写不出 = 没想清楚 / 该并页
- ② 表达式：____            ← 查《表达式词典》指到具体一类 + 选定表达（如"结构关系 → mermaid graph"）
- ③ 为什么：____            ← 为什么这个最清楚；对照词典红线自证。禁止理由"模板里有这个件"
- ④ 真实素材：____          ← 需要的真实图 / 数据；缺的标【需提供】
- 备注：____               ← 可选：分页取舍、与邻页关系、给 ppt-skills 的提示
```

---

## 填写须知

1. **四问缺一不可。** 任一空着 = 这页没设计完，别交。
2. **② 必须能在《表达式词典》里指到具体一类。** 指不到，多半是内容形态没想清。
3. **③ 禁止写"模板 / 库里有这个件"。** 理由要落在"这个表达对这页内容最清楚"。
4. **缺真实素材用【需提供】，不要用 `____` 占位。** `____` 代表"还没填"，【需提供】代表"已想清楚、待用户补料"。
5. 出稿前过一遍《SKILL.md 第 5 步 · 表达层自检》。
